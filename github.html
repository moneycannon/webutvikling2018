HVA ER GITHUB?

Github er en skylagringstjeneste for kode. Du kan her lagre kodene dine of fremgangen din. Og skulle du noensinne ønske å gå tilbake å starte tidligere i prosjektet så er det fullt mulig!. Du kan også dele kode og la flere jobbe på koden du lager. Dette er derfor det perfekte verktøy for deg som driver med kode alene eller sammen med flere.

Til høyre kan dere se min siste commit i Github.
